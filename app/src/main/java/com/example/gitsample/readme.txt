项目目录结构

algorithm               : 常见算法的实现
algorithm:link          : 链表
algorithm:sort          : 排序
algorithm:traversal     : 二叉树

anim                    : 各类动画实现

base                    : APP/常量

canvas                  : 各类通过 canvas 实现的控件展示
canvas:figure           : 通过 canvas 绘制几何图形
canvas:image            : 通过 canvas 绘制图片
canvas:path             : 通过 canvas 绘制路径
canvas:text             : 通过 canvas 绘制文案

function                : 一些小功能
function:applayout      : 通过 AppbarLayout 实现吸顶效果
function:appwidget      : 桌面组件实现
function:drag           : 拖拽效果实现
function:lock           : 九宫格图案解锁
function:screen         : 截屏功能
function:shortcut       : 桌面快捷方式
function:wallpaper      : 设置壁纸锁屏

main                    : 主界面

media                   : 媒体相关实现

system                  : 系统的一些属性列表
system:file             : 文件路径获取
system:info             : 手机系统/机型信息
system:kotlin           : Kotlin 的一些写法
system:permission       : 权限申请
system:screen           : 界面/全屏/状态栏展示

third                   : 第三方库的自己实现

widget                  : 一些好用的自定义控件
widget:dialog           : Dialog 的各种样子
widget:fragment         : Fragment 的各种展示
widget:text             : TextView 的各种展示
