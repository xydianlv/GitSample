项目目录结构

app                      : 业务模块儿

common                   : 支撑项目的基础模块儿集合
common:widget            : 一些自定义的测试组件
common:base              : 一些项目基础的常量/管理类
common:utils             : 一些工具类集合

function                 : 独立的功能模块儿
function:init            : 启动优化模块儿
function:list            : 实现 RecyclerView 数据与卡片注册绑定的控件

resource                 : 资源文件模块儿
